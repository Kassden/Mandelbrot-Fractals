# Three.js Mandelbrot

Interactive Mandelbrot viewer built with Next.js, Three.js, and a fullscreen fragment shader.

## What it is

- Fullscreen Mandelbrot render on a Three.js plane
- Drag to pan across the complex plane
- Scroll to zoom
- Live controls for iteration depth and palette offset

## Run it

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build check

```bash
npm run build
```
